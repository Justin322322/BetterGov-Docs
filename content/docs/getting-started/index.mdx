## Prerequisites

- Node.js (v22+)
- npm (v10+) 
- Git

## Quick Setup

```bash
# Clone your fork
git clone https://github.com/<your-username>/bettergov.git
cd bettergov

# Install and run
npm install
npm run dev
```

Visit `http://localhost:5173`
