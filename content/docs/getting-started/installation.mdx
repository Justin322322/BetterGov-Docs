## Local Development

```bash
# Clone and setup
git clone https://github.com/<your-username>/bettergov.git
cd bettergov
npm install

# Configure environment
cp .env.example .env

# Start development server
npm run dev
```

Visit `http://localhost:5173`

## Troubleshooting

**Port conflict:**
```bash
npx kill-port 5173
```

**Dependency issues:**
```bash
rm -rf node_modules package-lock.json
npm install
```
